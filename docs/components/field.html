Field

A set of components for building accessible, well-structured form fields.

Basic Field

Email 
<plank-field>
  <plank-field-label>Email</plank-field-label>
  <plank-input type="email" placeholder="email@example.com"></plank-input>
</plank-field>

With Description

Components

Component Description
plank-field Individual field wrapper with orientation support
plank-field-set Groups related fields together
plank-field-legend Title for a fieldset
plank-field-group Container for grouping multiple fields
plank-field-content Content area within a field
plank-field-label Label for a field input
plank-field-title Non-label title for display
plank-field-description Help text for a field
plank-field-separator Visual divider with optional text
plank-field-error Error message display

Attributes

Component Attribute Values Default
plank-field orientation "vertical" | "horizontal" | "responsive" "vertical"
plank-field data-invalid "true" | "false" -
plank-field-legend variant "legend" | "label" "legend"
